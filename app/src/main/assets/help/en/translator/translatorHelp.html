Help - Translation screen

Translation screen enables you to edit subtitle text (picture 1).


Picture 1. Translation screen

Translation screen contains three main sections:

  • Title bar - displays the name of currently opened subtitle file, and the button for displaying the menu.
  • Central area - shows current subtitle line.
  • Input field and buttons

Title bar

Displays the name of the currently opened subtitle file, as well as the button for showing the main menu. Menu contains three options - to save all changes made to the subtitle file, and to access translation screen settings and help.


Picture 2. Translation screen menu options

Central area

Shows three subtitle lines - the current one (in the middle, larger), previous and the following one. You can skip to before/after by clicking on corresponding lines.

Input field and buttons

Input field allows you to change the current subtitle line. Usually shows the text of the line being changed until the user inputs something.


Picture 3. Inputting text in portrait orienatation

In landscape orientation, only the input field will be shown on smaller screens. In this case, the grayed out current line text allows you to translate without hiding the virtual keyboard.


Picture 4. Inputting text in landscape orientation

Three buttons under the input field have the following role:

  • Copy Original - Copies the current line into the input field. Convenient if you want only to make small changes to the line.
  • Commit - changes the current line with text that was inputted into the input filed.
  • Commit & next - changes current line and switches over to the next one.
